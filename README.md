# Virtual Machine Manage

Basic virtual machine management, meant for developers to manage UTM, Vmware Fusion and Virtualbox VM's.  Script can be used to automate local developer Jenkins processes, pipelines, etc.


Related jenkins process automation repo:  https://github.com/clockworksspheres/jenkinsTools

For managing local VM's to assist CI/CD processes - can manage UTM, VMware Fusion and VirtualBox VM's on macOS.  Plans include to manage VMware Workstation, Virtualbox and HyperV VM's on Windows.  The tool is meant for developers, scientists, engineers and others to automate local CI/CT processes, that may feed into a CD process as well.
